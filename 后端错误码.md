#后端错误码

> Author : 胡子昂

- 此文档描述后端可能发生的错误和传输的错误码。

1. User

| 错误码 |          错误信息          |
| :----: | :------------------------: |
|   1    | 数据库学生信息插入出现错误 |
|   2    | 数据库学生信息更新出现错误 |
|   3    | 数据库学生信息查找出现错误 |
|   4    | 数据库学生信息删除出现错误 |
|   5    |       学生信息不存在       |

2. Seat


| 错误码 |          错误信息          |
| :----: | :------------------------: |
|  101   | 数据库座位信息插入出现错误 |
|  102   | 数据库座位信息更新出现错误 |
|  103   | 数据库座位信息查找出现错误 |
|  104   | 数据库座位信息删除出现错误 |
|  105   |        不存在该座位        |

| 错误码 |       错误信息       |
